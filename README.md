# Treinamento de Redes Convolucionais (Chest X-Ray)

## Descrição

Este projeto tem como objetivo realizar o treinamento de modelos de Redes Neurais Convolucionais (CNNs) para a classificação de imagens de radiografias de tórax. O foco principal é a detecção e diferenciação entre classes relacionadas a condições pulmonares (COVID-19, Pneumonia e Normal).
